# UI-гайд — Онлайн-кинотеатр

## Цвета

| Переменная | HEX | Назначение |
|---|---|---|
| `--bg` | `#141414` | Фон страницы |
| `--surface` | `#1f1f1f` | Фон карточек, форм, навбара |
| `--accent` | `#e50914` | Кнопки, ссылки, активные элементы |
| `--accent-hover` | `#b20710` | Hover-состояние акцента |
| `--text` | `#ffffff` | Основной текст |
| `--text-muted` | `#a3a3a3` | Вспомогательный текст, лейблы |
| `--border` | `#2a2a2a` | Границы карточек, инпутов |

## Шрифт

```css
font-family: Inter, Arial, sans-serif;
font-size: 16px;
line-height: 1.5;
```

| Элемент | Размер | Жирность |
|---|---|---|
| H1 | 24px | 700 |
| H2 | 18–22px | 600–700 |
| Основной текст | 16px | 400 |
| Вспомогательный текст | 14px | 400 |
| Мелкий текст | 13px | 400 |

## Компоненты

### Навбар
- Высота: 60px
- Фон: `#000`
- Граница снизу: `1px solid var(--border)`
- Логотип: акцентный цвет, 20px, 700

### Кнопки

**Основная:**
```css
background: #e50914;
color: #ffffff;
padding: 10px 20px;
border-radius: 4px;
font-size: 14px;
width: 100%;
```

**Вторичная:**
```css
background: transparent;
border: 1px solid var(--border);
color: var(--text-muted);
```

**Hover:** `background: #b20710`

**Focus:** `outline: 2px solid #e50914; outline-offset: 2px`

### Инпуты и селекты
```css
background: #1f1f1f;
border: 1px solid #2a2a2a;
border-radius: 4px;
padding: 10px 14px;
color: #ffffff;
font-size: 14px;
```
**Focus:** `border-color: #e50914`

### Карточки (.card)
```css
background: #1f1f1f;
border: 1px solid #2a2a2a;
border-radius: 8px;
padding: 24px;
```

### Таблицы
- Шапка: цвет `var(--text-muted)`, граница снизу `var(--border)`
- Строки: граница снизу `var(--border)`
- Последняя строка: без границы

### Ошибки
```css
color: #e50914;
font-size: 13px;
```

### Toast-уведомления
```css
position: fixed;
bottom: 24px;
right: 24px;
background: #16a34a;
color: #ffffff;
padding: 12px 20px;
border-radius: 8px;
font-size: 14px;
```

## Слайдер

- 4 слайда (постеры фильмов из БД)
- Автопрокрутка: 3 секунды
- Переход: `opacity 0.6s ease`
- Кнопки: круглые, 44×44px, `rgba(0,0,0,0.6)`
- Точки навигации: активная — акцентный цвет + ширина 20px, неактивная — `rgba(255,255,255,0.4)` + 8px
- Высота: 500px
- Фон: размытый постер (`blur(24px) brightness(0.3)`)
- Контент: постер слева + описание справа, по центру
